selling products, seamlessly integrating data from Excel files.

Employed advanced visualizations and interactive features to provide
actionable insights for optimizing sales strategies and inventory
management.

Leveraged expertise in data modeling and DAX function to deliver a
user-friendly solution facilitating informed decision-making processes.
